      🖨️ 完整旅程 PDF

      一鍵輸出：成員名單、預算、逐日行程＋過程日誌、記帳分帳結算、攜帶清單（瀏覽器列印 → 另存為 PDF），旅行結束印一份就是完整遊記。

      📅 行事曆 .ics

      下載後雙擊匯入 Google / Apple / Outlook 行事曆，每日行程自動建立事件（含交通建議）。

      📊 試算表 CSV

      含預算明細、行程、清單，可用 Excel / Google 試算表開啟。

      📋 複製行程文字（LINE 分享）

      把逐日行程＋預算摘要變成漂亮的文字訊息，直接貼到 LINE 群組給親朋好友。

      🤝 共編行程包（輪流編輯）

      把這趟行程（含成員、航班住宿、票券、記帳、日誌）打包傳給旅伴；對方匯入後修改，再傳回來給你匯入——輪流接力編輯。匯入前會自動備份，可救回。

      💾 旅程紀錄檔（備份 / 還原）

      完整 JSON 紀錄檔：行程、記帳、日誌、照片全打包。旅行結束存一份封存，下次規劃同地點時匯入參考。

      🗑️ 清除全部、重新開始

      一鍵清空所有目的地的行程、航班住宿、票券、記帳、日誌、照片與設定，回到全新狀態重新規劃。清除前會自動在本機備份一份，可按「還原上次清除」救回一次。重要資料請先用左方「下載紀錄檔」另存。

      👨‍👩‍👧 怎麼讓朋友也使用這套系統？

      • ① 傳網址（最推薦）：用上方「複製網址」把連結傳到 LINE／群組。朋友點開就能用，資料存在他自己的手機，跟你的完全分開。你之後更新系統，他下次開啟就是新版。
      • ② 加到主畫面更好用：朋友開啟網址後，手機瀏覽器選單按「加入主畫面」，會像 App 一樣有圖示，開啟更快、資料更穩。
      • ③ 想保留資料／換手機：到上方「💾 備份」存一份 JSON 檔；換裝置或不小心清掉資料時，用「還原」就能復原（還原前會先自動備份目前資料，可救回）。
      • ④ 分享你排好的行程：「📋 複製行程文字」貼到群組；或下載 .ics 讓大家各自匯入 Google／Apple 行事曆。
      • ⑤ 列印給長輩：「🖨️ 列印／存 PDF」只會印出總覽＋預算＋行程＋清單（已調整為黑字、會自動展開所有天數）。

      ※ 不建議把 .html 檔直接用 LINE 傳給朋友——iPhone 在 LINE 內開啟是「預覽模式」，勾選的清單關掉就消失。請改傳上方網址。
